Tendo o Visual Studio Code, junto à pasta do código.
Descompacte-o do .zip e no Visual Studio Code vá em file, open folder e escolha a pasta descompactada.
Instale a extenção Live Server no computador e execute o index.html em live server.
Crie uma conta ou faça o login em um email e senha criados por você, e a proveite o site de listas.
Exemplo funcional: EMAIL(jorge@gmail.com) , SENHA(123456).
